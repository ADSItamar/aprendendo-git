Estudando git e github.

Um ou vários arquivos podem estar em algum desses estados acima.

Untracked

Um arquivo encontra-se Untracked (NÃO RASTREADO) quando ele não foi selecionado para a area de staged

Staged

Para adicionar um arquivo a área de staged use o comando (git add nome_do_arquivo.html)

Unmodified / Modified

Um arquivo encrontra-se UNMODIFIED (nao modificado), a partir do momento que você altera ele passa para MODIFIED (modificado).

Assim que você terminar de fazer as alterações nos arquivos desejados, basta, usar o add para colocar os arquivos Modified na área de staged, para que você possa fazer o commit.
